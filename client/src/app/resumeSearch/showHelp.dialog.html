Known Operators

  • Operator AND
    Default implicit operator. Matches when both of its two arguments match.
    Example : java and python

  • Operator OR
    Matches when any of its two arguments match.
    Example: java or python

  • Operator NOT
    Matches when the first argument matches, but the second one does not. 
    Example: java not python 

  • Grouping operator (parentheses)
    The content within parentheses will have highest priority evaluation.
    Example: ( java or c++ ) and python

  • Phrase operator
    Matches when argument keywords match as an exact phrase.
    Example: " Multithreaded Programming "

Note- For auto suggestion to work properly, please provide space after any occurence of parentheses or quotes.
